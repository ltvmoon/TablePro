# AI Chat

Built-in AI chat panel for writing, explaining, and fixing SQL. Supports Claude, OpenAI, Gemini, OpenRouter, Ollama, and custom OpenAI-compatible endpoints.

## Opening the Panel

Press `Cmd+Shift+L`, click the AI toolbar button, or go **View** > **Toggle AI Chat**.

## Chatting with AI

Type a question and press Enter. The AI receives your database type, schema, and current query, then responds with syntax-highlighted SQL code blocks. Each code block has **Copy** and **Insert to Editor** buttons. Token counts appear below each response.

## Configuring AI Providers

Configure providers in **Settings** (`Cmd+,`) > **AI**. Ollama auto-detects on launch.

### Adding a Provider

Click **Add Provider**, choose a type, enter your API key, specify a model (or click **Fetch** to list available models), then **Test** to verify.

  <img

API keys are stored in macOS Keychain, never in plain text.

### Feature Routing

Route each feature (Chat, Explain, Optimize, Fix Error, Inline Suggestions) to a specific provider, or use the default.

Configure context in **Settings** > **AI** > **Context**: schema (on), current query (on), query results (off), max tables (20). Set a per-connection AI policy (Use Default, Always Allow, Ask Each Time, or Never) for sensitive connections.

Right-click SQL in the editor for **Explain with AI** and **Optimize with AI**.

Shortcuts: Toggle AI Chat (`Cmd+Shift+L`), Explain (`Cmd+L`), Optimize (`Cmd+Option+L`). Customizable in **Settings** > **Keyboard** > **AI**.

### Ask AI to Fix Errors

Query error dialogs include an **Ask AI to Fix** button. It opens the chat panel with the failed query and error message.
